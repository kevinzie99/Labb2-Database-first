# Music Console Application

This is a console application built with C# and Entity Framework Core connected to the Everyloop Music database schema.

The application allows users to manage playlists and tracks using full CRUD functionality.

---

## Features

- List playlists with tracks
- Create new playlists
- Add tracks to playlists
- Update playlist names
- Add and remove tracks from playlists
- Delete one or multiple playlists

---

## Technologies Used

- C#
- .NET
- Entity Framework Core
- SQL Server
- Visual Studio

---

## How To Run The Project

1. Clone the repository from GitHub

2. Open the solution file in Visual Studio

3. Make sure SQL Server is running

4. Update the connection string in `MusicContext` if needed

5. Build the solution

6. Run the application using the Start button or by pressing `F5`

---

## Database

The application uses the existing `music` schema from the Everyloop database.

Main tables used:

- playlists
- playlist_track
- tracks
